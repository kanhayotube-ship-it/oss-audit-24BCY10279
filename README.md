# OSS Audit Project

## Description
This project demonstrates basic shell scripting using Bash. It includes scripts for system information, Python environment checking, directory auditing, log analysis, and a user manifesto generator.
All scripts were tested successfully on macOS environment.

## Scripts Included

### script1.sh
Displays system information such as kernel version, user, home directory, uptime, and date.

### script2.sh
Checks Python installation and displays version and executable path.

### script3.sh
Performs directory audit by listing permissions of important directories.

### script4.sh
Checks system logs and searches for error entries.

### script5.sh
Generates a simple user manifesto by asking interactive questions and saving responses.

## Tools Used
- Bash (Shell Scripting)
- macOS Terminal
- GitHub

## How to Run 
chmod +x script1.sh
./script1.sh
